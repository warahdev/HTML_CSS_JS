03 - Flexbox

Simplifica o alinhamento e a distribuição do espaço entre os itens de um contêiner.

  • Contêiner: elemento pai que recebe a propriedade display: flex;.
  • Itens: elementos filhos diretos do contêiner.

Propriedades do contêiner

Propriedade flex-direction

Define a direção do eixo principal.

  • row (valor padrão)
  • row-reverse
  • column
  • column-reverse

Exemplo: flex-direction: column

Propriedade justify-content

Alinha os itens flexiveis pelo eixo principal.

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly

Exemplo: com flex-flow: row wrap; e justify-content: space-between;
