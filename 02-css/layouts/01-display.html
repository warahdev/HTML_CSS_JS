1 - Display

O display é uma propriedade importante do CSS para controlar o layout. Determina se um elemento HTML é tratado como um bloco ou um elemento em linha.

Block

Exemplo de elementos com o valor padrâo display: block;.

Hello Kitty com seu laço vermelho a brilhar,

Espalha alegria onde quer que vá chegar.

Inline

Exemplo de elementos com o valor padrâo display: inline;.

Com sorriso doce, coração de algodão, Ensina ao mundo a força da união.
